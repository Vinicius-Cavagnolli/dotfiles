## ⚙️ Installation

```bash
git clone https://github.com/Vinicius-Cavagnolli/dotfiles.git ~/.dotfiles-bkp
cd ~/.dotfiles-bkp
chmod +x install.sh
./install.sh
```

> ⚠️ The script will prompt for `sudo` to start some utilities.

## ✅ Dependencies

Make sure you have the following installed:

- `git`
- `curl`
- `npm`
- `vim`
- `zsh`
